Technische Sicherheitstipps

Helfen Sie allen, sicher zu bleiben

Sie müssen online (und offline) sicher bleiben. Bitte leiten Sie diese Liste an alle Angehörigen weiter, die anfällig für Betrügereien oder Phising-Attacken sein könnten.

E-Mail | Textnachrichten (SMS) | Online/Webseiten | Telefon | In-Person

E-Mail

  • E-Mail ist KEIN sicheres Medium. Das bedeutet:
    • Senden Sie nichts per E-Mail, von dem Sie nicht möchten, dass ein Hacker es sieht, einschließlich:
      • Sozialversicherungsnummer
      • Kreditkartennummer
      • Führerschein
      • Benutzername/Passwort-Kombination zu JEDEM Konto
      • Fotos von Ausweisen
      • PDFs oder Dateien, die eine der oben genannten Informationen enthalten
    • Wenn Ihnen jemand etwas mit sensiblen Informationen schickt, sollten Sie, wenn Sie antworten, diese Informationen unbedingt löschen (normalerweise unter Ihrer Antwort)
  • Vermeiden Sie das Anklicken von Links in E-Mails.
  • Wenn Sie eine E-Mail von JEMANDEM erhalten, in der Sie aufgefordert werden, auf einen Link zu klicken, um sich anzumelden, KLICKEN SIE NICHT DARAUF. Öffnen Sie ein neues Browser-Fenster und geben Sie die URL der Firma manuell ein und melden Sie sich selbst an.
  • Jede E-Mail kann weitergeleitet werden; senden Sie keine E-Mails, von denen Sie nicht möchten, dass sie Ihnen selbst zugeordnet werden.
  • Öffnen Sie keine Anhänge von anderen, auch nicht von Personen, die Sie kennen, es sei denn, Sie haben sie erwartet.
  • Wenn sich eine E-Mail auch nur im Entferntesten verdächtig anfühlt, vertrauen Sie Ihrem Bauchgefühl.

Textnachrichten (SMS)

  • Vermeiden Sie es, auf Links in Textnachrichten zu klicken, es sei denn, Sie haben damit gerechnet.
  • Geben Sie niemals jemandem einen Sicherheitscode, den Sie per SMS erhalten haben.

Online/Webseiten

  • Wenn Sie einen Dienst haben, der dies anbietet, schalten Sie die "Zwei-Faktor-Authentifizierung" (2FA) oder "Multi-Faktor-Authentifizierung" ein. Damit können Sie Ihr Telefon als zusätzliche Sicherheitsmaßnahme für die Anmeldung bei Konten verwenden.
  • Verwenden Sie niemals das gleiche Passwort für verschiedene Dienste. NIEMALS. Stattdessen...
  • Verwenden Sie einen "Passwort-Manager", um Ihre Passwörter sicher zu speichern.
  • Stellen Sie sicher, dass alle Kennwörter, die Sie regelmäßig verwenden, für Sie leicht zu merken, aber schwer zu erraten sind. Einige Tipps:
    • Verwenden Sie eine 4-5-Wort-Phrase, die Sie nie in einem Buch lesen oder als Liedtext hören würden
    • Zum Beispiel: Pfennig-Wasser-Lautsprecher-Laterne
  • Geben Sie Ihre Passwörter nicht an andere weiter.
  • Nur weil eine Website das -Symbol hat oder mit "https://" beginnt, bedeutet das nicht, dass sie die ist, die sie vorgibt zu sein. Überprüfen Sie den tatsächlichen Domainnamen (z. B. sparkasse.de und nicht sparkase.de).
  • Wenn Ihr Browser Ihnen sagt, dass Sie einen Virus haben, haben Sie keinen. Niemals.
  • Installieren Sie keine Browser-Erweiterungen, es sei denn, Sie benötigen sie und wissen, dass sie von einer vertrauenswürdigen Quelle stammen.

Telefon

  • Telefonnummern können "gefälscht" werden, das heißt jemand kann sich als jemand anderes ausgeben. Rufen Sie im Zweifelsfall unter einer Nummer zurück, die SIE kennen.
  • Wenn jemand von einem der folgenden Namen anruft, legen Sie auf. Die rufen Sie nicht an:
    • Apple
    • Microsoft
    • Google
    • BND
    • BKA
    • Jedes Tech-Unternehmen
  • Seien Sie vorsichtig, wer sich in Ihrer Nähe aufhält und Ihre Gespräche mithören kann, insbesondere wenn Sie sensible Informationen verwenden.

In-Person

  • Vermeiden Sie es, die folgenden Informationen aufzuschreiben, und wenn Sie es doch tun, sorgen Sie persönlich dafür, dass sie sofort vernichtet werden:
    • Sozialversicherungsnummer
    • Kreditkartennummer
    • Führerschein
    • Benutzername/Passwort-Kombination zu JEDEM Konto
  • Seien Sie vorsichtig, wer sich in Ihrer Nähe aufhält und Ihre Gespräche mithören kann, insbesondere wenn Sie sensible Informationen verwenden.
  • Lassen Sie Ihr Handy, Ihre Geldbörse oder Ihr Portemonnaie nicht draußen liegen
